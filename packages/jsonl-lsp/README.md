# @sarvagnan/jsonl-lsp

`jsonl-lsp --stdio` — a Language Server Protocol server for JSON Lines /
NDJSON: publishes per-record diagnostics and provides document/range
formatting. Works with any LSP-capable editor (BBEdit, Neovim, Helix, Zed,
Emacs, …). Built on `@sarvagnan/jsonl-core`. Part of [jsonl-tools](../../README.md).
